identeefi-postgres-database
	1. create the tables using the create_table.txt.
	2. fill in the tables with the data in the csv files.

identeefi-backend-node
	1. run `npm i`
	2. run `node postgres`

identeefi-frontend-react
	1. run `npm i` to download dependencies
	2. run `npm start` to run the program locally.

identeefi-smartcontract-solidity
	- This smart contract has been deployed to the Sepolia Testnet, you just need to make sure you have the MetaMask wallet to connect to the blockchain to run this code. This code is uploaded here for references purposes only.
